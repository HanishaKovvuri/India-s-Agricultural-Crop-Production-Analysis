# 🌾 India's Agricultural Crop Production Analysis

## 📖 Project Overview

India's Agricultural Crop Production Analysis is a Tableau-based Business Intelligence project that analyzes historical agricultural production data from **1997 to 2021**. The project transforms raw agricultural data into interactive dashboards and stories, enabling users to explore crop production trends, seasonal variations, regional performance, and year-over-year growth.

The project helps stakeholders such as farmers, government policymakers, researchers, and agribusiness investors make informed decisions through data-driven insights.

---

## 🎯 Objectives

- Analyze agricultural crop production across India.
- Study state-wise, crop-wise, and season-wise production trends.
- Identify high-performing crops and regions.
- Visualize long-term agricultural production patterns.
- Support data-driven decision-making using interactive dashboards.
